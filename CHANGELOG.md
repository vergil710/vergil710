# Changelog

所有重要变更都会记录在这个文件中。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本按倒序排列。

## [Unreleased]

## [0.5.0] - 2026-06-09

### Added

- 建立固定维护规则：后续每次重要修改都需要同步更新 `CHANGELOG.md`。
- 完善 `README.md`，说明项目模块、重要目录、内容编辑方式、本地开发和 GitHub Pages 发布流程。

### Fixed

- 修复文章页下滑后触发返回顶部按钮时 Hook 顺序变化导致的客户端崩溃。
- 稳定文章封面 URL 拼接逻辑，避免依赖浏览器全局 `origin` 变量。

## [0.4.0] - 2026-06-09

### Fixed

- 修复 Live2D 在 GitHub Pages 子路径部署下加载失败的问题，模型入口现在会自动适配 `/vergil710` base path。
- 优化 Live2D 错误提示：运行库加载失败时给出更明确的提示。

## [0.3.0] - 2026-06-09

### Fixed

- 修复图片墙图片在 GitHub Pages 子路径下无法显示的问题。
- 修复首页图片设置和背景图片设置中的本地图片预览路径。

## [0.2.0] - 2026-06-08

### Added

- 新增任务清单页面，支持新增、勾选、修改、删除待办。
- 任务清单支持自定义标签，并可按标签筛选。
- 同步论文写作、AI 人设和研究任务相关内容到博客文章模块。

### Changed

- 将首页点赞从外部接口改为浏览器本地计数，避免依赖模板作者的服务。
- 整理博客主要模块为论文写作、AI 人设、任务清单和个人内容展示。

### Fixed

- 修复任务清单页面使用模板 `card` 样式导致模块重叠的问题。
- 清理重复生成的 `docs/* 2*` 发布产物。

## [0.1.0] - 2026-06-08

### Added

- 基于 `YYsuni/2025-blog-public` 模板初始化个人博客。
- 配置 GitHub Pages 静态发布目录 `docs/`。
- 设置站点名称、描述、主题色、社交链接和首页内容。
- 建立 Markdown 文章目录结构。
