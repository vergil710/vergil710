控制类论文实验设计：指标、对比和消融

从论文主张出发设计仿真实验，而不是先画图再解释。

2026-06-03
阅读时长: 1 分钟

控制类论文的实验设计不能只问“画哪些图”，而要先问“我的论文主张是什么”。每一个实验都应该服务一个主张。

1. 指标

如果论文主张是节能准时速度曲线规划，常见指标包括：

  • 到站时间误差；
  • 总牵引能耗；
  • 最大速度约束违反量；
  • 舒适度或冲击率；
  • 计算时间；
  • 曲线跟踪误差。

指标要和问题对应，不能只挑对自己有利的图。

2. 对比方法

对比方法至少应该覆盖：

  • 人工驾驶或经验策略；
  • 传统优化方法；
  • 现有基线控制器；
  • 去掉关键模块后的版本。

如果没有合理对比，读者很难判断改进来自方法本身，还是来自场景设置。

3. 消融实验

消融实验回答的是：每个模块到底有没有贡献。

例如：

  • 去掉在线参数识别；
  • 去掉安全约束层；
  • 固定规划步长；
  • 不考虑分相区；
  • 不使用 warm start。

每个消融都应该对应一个设计选择。

4. 鲁棒性实验

真实系统有扰动和不确定性，所以还要看：

  • 坡度误差；
  • 牵引制动参数偏差；
  • 临时限速变化；
  • 计划时间扰动；
  • 不同线路和不同车型。

好的实验不是把图画满，而是让读者相信：方法在关键场景里确实支撑了论文主张。
